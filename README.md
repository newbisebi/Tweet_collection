## Instructions 
Create a file "AuthTwitterCodes.txt" at project root with 4 lines corresponding to Twitter API keys (to be created in your developper account on Twitter), in that order :
- app_key 
- app_secret
- oauth_token 
- oauth_token_secret
No space, no comma.
